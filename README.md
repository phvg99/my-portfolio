# Portfolio

Built with next.js, [shadcn/ui](https://ui.shadcn.com/), and [magic ui](https://magicui.design/), deployed on GitHub Pages.

# Features

- Setup only takes a few minutes by editing the [single config file](./src/data/resume.tsx)
- Built using Next.js 14, React, Typescript, Shadcn/UI, TailwindCSS, Framer Motion, Magic UI
- Includes a blog
- Responsive for different devices
- Optimized for Next.js and GitHub Pages

# Getting Started Locally

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/phvg99/portfolio
   ```

2. Move to the cloned directory

   ```bash
   cd portfolio
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the local Server:

   ```bash
   npm run dev
   ```

5. Open the [Config file](./src/data/resume.tsx) and make changes

# Deployment to GitHub Pages

1. Create a new GitHub repository for your portfolio

2. Push your code to the repository:
   ```bash
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/phvg99/portfolio.git
   git push -u origin main
   ```

3. Configure GitHub Pages:
   - Go to your repository on GitHub
   - Navigate to Settings > Pages
   - Under "Source", select "GitHub Actions"
   - Your site will be deployed automatically when you push to the main branch

4. After deployment, your site will be available at:
   `https://phvg99.github.io/portfolio`

5. To use a custom domain:
   - Add your domain in GitHub repository settings
   - Create a CNAME file in the `/public` directory with your domain name
   - Update DNS settings with your domain provider
